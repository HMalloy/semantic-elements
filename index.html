Here is the list of items completed for my homework:

  1. Read the following articles:
    • An Overview of HTML5 semantics
    • HTML5 Semantic Elements
    • List of Possible HTML Elements
  2. Write one paragraph summarizing what semantic elements are (max of 150 words)
    • Semantic elements are code elements whose names clearly represent the content that is contained within them. They enhance searchability, accessibility, internationalization, interoperability, and they help other developers read and understand your code. Semantic elements were introduced with HTML5 and were informed by analyzing div names from 1 billion pages.
  3. Create a new git repository on your computer. Name it semantic-elements.
  4. Inside the repo, code an HTML document. Name it index.html.
  5. Use at least 15 different types of HTML elements (correctly!)*
  6. View your document in a browser to ensure it displays as you expected.
  7. Push your code to GitHub as a new repository.

* Is it ok to use checkbox inputs within list items? Does this fall into the realm of being used correctly? The code validator didn't throw any errors, but perhaps this is not a good overall practice?
